What Is ColorFile ?

ColorFile is an encoding system that turns any file into a .png
Bigger the file, Bigger the .png !
Dont Worry! you can turn them back into original form. (As long as filename stays the same)

Ok, but why ?

Maybe it is a nice thing to have ?
Seeing your file encoded into a image of different colored pixels must be nice !
By encoding your files, you can save your files in any free image upload file and free up some space !
(I hope this is not abusing free site for free storage :D )

How it works for the curious:

Each file in a computer is made of 0 and 1, for shortening it, we encode it.
But what is encoding ?
Each 00001111 combination is encoded and given a represenetive like "dG"
Then each of this keys are given a rgb value, then put into png.
Thats how it works. Yes, i made this encoding system because i was bored.
